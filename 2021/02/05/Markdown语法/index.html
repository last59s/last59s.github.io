Markdown语法

发表于 2021-02-05 更新于 2021-02-06 阅读次数： Valine：

不会吧不会吧，这年头居然还有人还不会写Markdown吧

没错正是在下

本页及本站的所有文章，均由Markdown，编写工具：Typora

标题

一共有六级标题，在你要写的标题前加上对应数目的 # ,且 # 和标题间要留有空格

强调

斜体：在要倾斜的文字左右分别用一个 * 包起来

粗体：在要加粗的文字左右分别用两个 * 包起来

加粗斜体：在要倾斜和加粗的文字左右分别用三个 * 包起来

删除线：在要加删除线的文字左右分别用两个 ~ 包起来

背景高亮：在要高亮的文字左右分别用两个 = 包起来

列表

- 无序列表

在你的列表前加上 * 或 - 或 + ，并使用空格分隔开

  • 无序
  • 列表
  • 没了

- 有序列表

比无序列表相对简单，阿拉伯数字加上英文句点再加空格分隔你的列表，值得注意的是，有序列表的序号是根据第一行列表的数字顺序来的，也就是说顺序无法打乱

  1. one
  2. two
  1. three
  2. six
  3. four

代码

如果要标记一小段行内代码，可以直接用 ` 将代码包起来

不过正经人谁还写行内代码呀，直接用两行 ``` 把代码块包起来它不香嘛

引用

在引用的文字前加 > 即可。引用也可以嵌套，如加两个 >> 三个 >>> n个…
貌似可以一直加下去，但没什么卵用

表格

这个的格式特别简洁明了，写起来相对简单，也不用在意代码是否对齐

分割线

三个或者三个以上的 - 或者 * 或者 _ 都可以，不过显示效果都是一样的

链接

支持2种链接方式：行内式和参数式，不管是哪一种，链接文字都是用 [ ] 来命名

- 行内式

将链接的名字放在 [ ] 中，链接地址放在随后的 ( ) 中，也可以添加引号 ‘ ‘,对其注释

404大厂 百度

- 参数式

在链接文字的方括号后面再接上另一个方括号，而在第二个方括号里面要填入用以辨识链接的标记

其中包含链接的标记可以放置在文章的其他地方

b站链接Bilibili

图片

可以插入网络的图片，也可以插入本地的图片，还能把图片存入文章里

- 网络图片

和链接的格式相似，仅仅多了一个 ! ,可以用参数式，也可以用行内式。

- 本地图片

跟上面的网络图片格式差不多，不同的是图片的位置路径，支持绝对路径和相对路径

- Base64图片

用base64转码工具把图片转成一段字符串，然后把字符串填到基础格式中链接的那个位置

不过我不推荐这种方式，会增大文件的体积，还不如通过外链读取来的快呢

公式

可以插入数学公式和一些流程图，就像这样$\int\frac{dx}{\sqrt{1-x^2}}=\arctan x+C$

至于语法嘛，也不难，就是 Latex 的语法，不过记得在编辑器中先打开扩展语法的选项才能实时预览

另外，本站采用的是 valine 评论系统，也支持 markdown 的语法，至于支持的程度有多少，我就不得而知啦，所以欢迎各位在下方或在留言板中评论
